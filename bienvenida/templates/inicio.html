Seguridad CSRF Implementada

Este sistema implementa protección CSRF (Cross-Site Request Forgery) en todas las operaciones que modifican datos (crear, actualizar, eliminar). Esto garantiza que las peticiones maliciosas desde otros sitios web no puedan realizar operaciones no autorizadas.
